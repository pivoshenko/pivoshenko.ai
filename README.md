# pivoshenko.ai

Central workspace for AI agent operations.

## Scope

- MCP configurations
- Agent Skills
- Prompt templates
- Tooling scripts
- Automation runbooks
- Experiments and notes

## Structure

- `mcp/` - MCP configs and server profiles
- `skills/` - reusable agent skills
- `prompts/` - prompt packs and templates
- `scripts/` - helper scripts and automations
- `docs/` - architecture notes and runbooks
